# Snakes-and-Ladders-Game
Snakes and Ladders is a multiplayer game designed in Python 3.9

To play the game :
Open game.py and run it.

Models -
  Snakes,
  Ladders,
  Players,
  Board,
  Entities.
  
Features -
No. of Players : n (any number of players).

Flexible entities,
Scalable and Reusable Code.
